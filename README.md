# Semantic Components

A collection of accessible, customizable UI components for Angular applications.

## Features

- Built with Angular 21+ and standalone components
- Fully accessible (WCAG AA compliant)
- Styled with Tailwind CSS
- Includes components: Alert Dialog, Breadcrumb, Carousel, Chart, Slider, Toggle, Toggle Group, and more

## Development

This project uses [Nx](https://nx.dev) as the build system.

### Prerequisites

- Node.js
- npm or yarn

### Getting Started

```bash
# Install dependencies
npm install

# Run the showcase app
npx nx serve showcase

# Run tests
npx nx test

# Build the library
npx nx build
```
